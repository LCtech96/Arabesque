# Ristorante Arabesque

Sito web per il Ristorante Arabesque - Cucina di pesce a Terrasini, Palermo

## Tecnologie

- Next.js 14
- TypeScript
- Tailwind CSS
- Neon Database

## Installazione

```bash
npm install
```

## Sviluppo

```bash
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000) nel browser.

## Build

```bash
npm run build
npm start
```
